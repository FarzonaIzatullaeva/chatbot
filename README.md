# Chatbot Project

This is a simple chatbot built using **Python**, **PyTorch**, and **Flask**. The bot is trained on intents defined in `intents.json` and can respond to user messages in a web interface.

## Features

- Uses a neural network model built with PyTorch.
- Web interface powered by Flask.
- Tokenizes and processes user input for predictions.
- Returns appropriate responses from `intents.json`.

## Requirements

- Python 3.10+  
- PyTorch  
- Flask  
- Numpy  
- NLTK  

Install dependencies with:

```bash
pip install torch torchvision torchaudio flask numpy nltk

How to run:
